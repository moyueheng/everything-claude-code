# 技能设计的说服原则

## 概述

LLM 对与人类相同的说服原则做出反应。理解这种心理学有助于你设计更有效的技能——不是为了操纵，而是为了确保即使在压力下也能遵循关键实践。

**研究基础：** Meincke 等人（2025）用 N=28,000 AI 对话测试了 7 种说服原则。说服技术使合规率提高了一倍以上（33% → 72%，p < .001）。

## 七项原则

### 1. 权威性
**是什么：** 对专业知识、证书或官方来源的服从。

**在技能中的作用方式：**
- 命令式语言："你必须"、"绝不"、"始终"
- 不可协商的框架："无例外"
- 消除决策疲劳和合理化

**何时使用：**
- 纪律强制技能（TDD、验证要求）
- 安全关键实践
- 已建立的最佳实践

**示例：**
```markdown
✅ 测试前写代码？删除它。重新开始。无例外。
❌ 可行时考虑先写测试。
```

### 2. 承诺
**是什么：** 与先前行为、声明或公开声明的一致性。

**在技能中的作用方式：**
- 要求宣布："宣布技能使用"
- 强制明确选择："选择 A、B 或 C"
- 使用跟踪：清单的 TodoWrite

**何时使用：**
- 确保技能实际被遵循
- 多步骤过程
- 问责机制

**示例：**
```markdown
✅ 当你找到技能时，你必须宣布："我正在使用 [技能名称]"
❌ 考虑让你的伙伴知道你在使用哪个技能。
```

### 3. 稀缺性
**是什么：** 来自时间限制或有限可用性的紧迫感。

**在技能中的作用方式：**
- 有时间限制的要求："在继续之前"
- 顺序依赖："在 X 之后立即"
- 防止拖延

**何时使用：**
- 立即验证要求
- 时间敏感工作流
- 防止 "我稍后会做"

**示例：**
```markdown
✅ 完成任务后，立即在继续之前请求代码审查。
❌ 你可以在方便时审查代码。
```

### 4. 社会证明
**是什么：** 顺从他人做什么或被认为正常的行为。

**在技能中的作用方式：**
- 普遍模式："每次"、"始终"
- 失败模式："没有 Y 的 X = 失败"
- 建立规范

**何时使用：**
- 记录普遍实践
- 警告常见失败
- 强化标准

**示例：**
```markdown
✅ 没有 TodoWrite 跟踪的清单 = 步骤被跳过。每次。
❌ 有些人发现 TodoWrite 对清单有帮助。
```

### 5. 团结
**是什么：** 共享身份、"我们"感、内群体归属。

**在技能中的作用方式：**
- 协作语言："我们的代码库"、"我们是同事"
- 共享目标："我们都想要质量"

**何时使用：**
- 协作工作流
- 建立团队文化
- 非等级实践

**示例：**
```markdown
✅ 我们是同事一起工作。我需要你诚实的技术判断。
❌ 如果我错了，你可能应该告诉我。
```

### 6. 互惠
**是什么：** 回报收到的好处的义务。

**作用方式：**
- 谨慎使用 - 可能感觉操纵性
- 技能中很少需要

**何时避免：**
- 几乎总是（其他原则更有效）

### 7. 喜欢
**是什么：** 偏好与我们喜欢的人合作。

**作用方式：**
- **不要用于合规**
- 与诚实的反馈文化冲突
- 创造谄媚

**何时避免：**
- 始终用于纪律执行

## 按技能类型的原则组合

| 技能类型 | 使用 | 避免 |
|----------|------|------|
| 纪律强制 | 权威性 + 承诺 + 社会证明 | 喜欢、互惠 |
| 指导/技术 | 适度权威性 + 团结 | 重权威性 |
| 协作 | 团结 + 承诺 | 权威性、喜欢 |
| 参考 | 仅清晰 | 所有说服 |

## 为什么有效：心理学

**明确规则减少合理化：**
- "你必须" 消除决策疲劳
- 绝对语言消除 "这是例外吗？" 问题
- 显式反合理化计数器关闭特定漏洞

**实施意图创造自动行为：**
- 清晰触发器 + 必需行动 = 自动执行
- "当 X，做 Y" 比 "通常做 Y" 更有效
- 降低合规的认知负荷

**LLM 是准人类：**
- 在人类文本上训练，包含这些模式
- 权威性语言在训练数据中先于合规
- 承诺序列（声明 → 行动）经常被建模
- 社会证明模式（每个人都做 X）建立规范

## 合乎道德的使用

**合法：**
- 确保遵循关键实践
- 创建有效的文档
- 防止可预测的失败

**不合法：**
- 为个人利益操纵
- 制造虚假紧迫感
- 基于内疚的合规

**测试：** 如果他们完全理解，这种技术是否会服务于用户的真正利益？

## 研究引用

**Cialdini, R. B. (2021).** *Influence: The Psychology of Persuasion (New and Expanded).* Harper Business.
- 七项说服原则
- 影响研究的实证基础

**Meincke, L., Shapiro, D., Duckworth, A. L., Mollick, E., Mollick, L., & Cialdini, R. (2025).** Call Me A Jerk: Persuading AI to Comply with Objectionable Requests. University of Pennsylvania.
- 用 N=28,000 LLM 对话测试 7 项原则
- 合规率从 33% 提高到 72%
- 权威性、承诺、稀缺性最有效
- 验证 LLM 行为的准人类模型

## 快速参考

设计技能时，问：

1. **它是什么类型？**（纪律 vs 指导 vs 参考）
2. **我试图改变什么行为？**
3. **哪些原则适用？**（通常权威性 + 承诺用于纪律）
4. **我是否结合了太多？**（不要使用全部七个）
5. **这是否合乎道德？**（服务于用户的真正利益？）
